Blooket Hacker

Instructions

Thanks for downloading Blooket Hacker!


How to use:
Open the extension popup window by clicking the Blooket Hacker icon in the extensions menu.
When on the Blooket game tab, click the check button to start the program on that tab.
Then, begin playing the game. The extension will automatically skip past unnecessary screens to allow you to answer as many questions as fast as possible. The AI will learn correct answers from you and next time the questions show up, it will answer it for you.


All Features:
  • AI learns correct answers from gameplay
  • Answers questions for you automatically
  • Skips past unnecessary screens
  • Previews all locked blooks

Game Specific:
  • Hides all sabotages (Fishing Frenzy)
  • Remembers passwords (Crypto Hack)
  • Calculates most efficient upgrade (Factory)
